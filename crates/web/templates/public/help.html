Any language that compiles to a Linux x86_64 static binary (Metal) or a WASI WebAssembly module (Liquid). Go, Rust, Zig, C, and C++ work out of the box. Install the CLI via Homebrew (brew install liquidmetal/tap/flux) or the install script (curl -fsSL https://get.liquidmetal.dev | sh). The CLI auto-detects Go and Rust projects and configures build commands automatically.
Two different engines for different workloads.

Metal gives you a dedicated VM on bare metal hardware. Always on, fixed monthly price, real Linux kernel, filesystem, and networking. Best for production APIs, databases, background jobs — anything that needs a real server.

Liquid is serverless WebAssembly. Scale to zero, wake on request, pay per invocation. Sub-millisecond cold starts. Best for lightweight APIs, webhooks, and data transforms.
Metal VMs are billed at a fixed monthly rate based on dedicated vCPU tier: $17/mo (0.5 vCPU, 2GB RAM), $33/mo (1 vCPU, 4GB RAM), or $66/mo (2 vCPU, 8GB RAM). All tiers include NVMe storage and dedicated CPU — no shared resources.

Liquid is billed per invocation at $0.30 per 1M requests. Every account gets 1M free Liquid invocations per month. No credit card required to start.
Yes. Every account gets 1M free Liquid (Wasm) invocations per month. Liquid services scale to zero when idle. No credit card required. Metal VMs start at $17/mo.

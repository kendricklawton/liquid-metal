Any language that compiles to a Linux x86_64 static binary (Metal) or a WASI WebAssembly module (Liquid). Go, Rust, Zig, C, and C++ work out of the box. Install the CLI via Homebrew (brew install liquidmetal/tap/flux) or the install script (curl -fsSL https://get.liquidmetal.dev | sh). The CLI auto-detects Go and Rust projects and configures build commands automatically.
Both are serverless — scale to zero, wake on request, pay per invocation.

Metal runs your code in a Firecracker microVM restored from snapshot in <15ms. Real Linux kernel, filesystem, networking. Best for workloads needing syscalls, FFI, or native dependencies.

Liquid runs your code as a WebAssembly module in Wasmtime with <1ms cold starts. Sandboxed, no kernel. Best for APIs, webhooks, and data transforms.
Every tier includes 1M free Liquid invocations/month. Metal is billed from the first request (Pro and Team only). Hobby is free with a 5-service limit (Liquid only). Pro ($10/month) and Team ($20/month) include monthly credits and Metal access. Usage: Metal at $0.60/1M invocations + $0.0001/GB-sec compute, Liquid at $0.30/1M invocations. Same rates on every plan.
Yes. The Hobby tier is free forever with up to 5 Liquid services and 1M invocations/month. Services scale to zero after 5 minutes of inactivity. No credit card required.

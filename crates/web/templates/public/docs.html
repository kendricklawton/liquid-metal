// quickstart

01 / Install the CLI

Choose your install method:
# macOS / Linux (Homebrew)
brew install liquidmetal/tap/flux

# Shell script (Linux / macOS)
curl -fsSL https://get.liquidmetal.dev | sh

# From source (requires Rust toolchain)
cargo install flux
$ flux --help

02 / Authenticate

$
flux login
flux whoami

03 / Initialize a project

$
cd my-app
flux init --engine liquid

04 / Deploy

$
flux deploy
flux status
